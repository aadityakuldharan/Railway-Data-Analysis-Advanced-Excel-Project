# 🚆 Railway Data Analysis (Advanced Excel Project)



An **Advanced Excel project** analyzing railway data using pivot tables, charts, conditional formatting, and advanced Excel functions.  
This project demonstrates how Excel can transform raw data into actionable insights for decision-making.

## 📖 Project Overview
The project analyzes railway-related data to:
- Track train schedules and performance  
- Identify trends using **PivotTables & PivotCharts**  
- Apply **conditional formatting** to highlight key metrics  
- Build an interactive dashboard for better visualization  

This project is designed as part of **Advanced Excel learning** to showcase data cleaning, analysis, and visualization skills.

---

## ✨ Key Features
- 🗂️ **Data Cleaning & Formatting** – Removing duplicates, applying data validation  
- 📊 **PivotTables & PivotCharts** – Summarizing large datasets for insights  
- 🎨 **Conditional Formatting** – Highlighting delays, costs, or performance metrics  
- 🧮 **Advanced Functions** – VLOOKUP, INDEX-MATCH, IF, SUMIFS, COUNTIFS  
- 📈 **Interactive Dashboard** – Easy-to-read visuals of railway data metrics
